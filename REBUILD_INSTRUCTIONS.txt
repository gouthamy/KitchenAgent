🔨 REBUILD STEPS (In Xcode):

1️⃣  CLEAN BUILD FOLDER
   • Press: ⌘⇧K (Command + Shift + K)
   • Or: Product → Clean Build Folder
   • Wait for "Clean Finished"

2️⃣  SELECT DEVICE
   • Click device dropdown (top toolbar)
   • Select: "iPhone 15" (or any iOS 17+ simulator)
   • Make sure it says "iPhone 15" not "Any iOS Device"

3️⃣  BUILD PROJECT
   • Press: ⌘B (Command + B)
   • Or: Product → Build
   • Wait for "Build Succeeded" ✅

4️⃣  RUN APP
   • Press: ⌘R (Command + R)
   • Or: Product → Run
   • Simulator will launch automatically
   • App will install and open\! 🎉

❌ IF BUILD FAILS:

1. Check Error Message:
   ─────────────────────
   • Red errors will show in Xcode
   • Take a screenshot and share it
   • Most common: "Cannot find type..."

2. Quick Fixes:
   ────────────
   a) Clean Derived Data:
      • Xcode → Settings → Locations
      • Click arrow next to "Derived Data"
      • Delete "KitchenAgent" folder
      • Retry build

   b) Check Target Membership:
      • Select a file with error
      • Open File Inspector (right panel)
      • Check "KitchenAgent" under Target Membership

   c) Restart Xcode:
      • Close Xcode completely
      • Reopen project
      • Try again
